# <a name="add-and-compile-the-xbox-live-apis-source-in-your-uwp-project"></a>UWP 프로젝트에서 Xbox Live Api 소스 추가 및 컴파일

이제 Xbox Live API (XSAPI) 원본이 GitHub에서 사용할 수 있는 [https://github.com/Microsoft/xbox-live-api](https://github.com/Microsoft/xbox-live-api). 개발자가 로컬 빌드를 사용 하는 프로젝트를 업데이트 하려면 다음이 지침에 따라 수 있습니다.

하는 경우 XSAPI 직접 빌드 하려는 경우:
1. 오류 코드에서 제공 되는 위치를 이해 하는 문제를 디버그 해야 합니다.
1. 경우 QFE을 배포할 수 있는 것,는 문제를 해결 하려면 소스 코드 패치를 제공 합니다.

XSAPI 프로젝트를 직접 컴파일하려면 GitHub 프로젝트 사이트의 지침을 따릅니다.
