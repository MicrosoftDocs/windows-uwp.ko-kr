# <a name="media-capture-page-dev-home"></a>미디어 캡처 페이지(개발자 홈)
   
  
개발자 홈의 **미디어 캡처** 페이지에서는 현재 콘솔에서 실행 중인 타이틀의 비디오를 캡처합니다.   
 ![캡처 개발자 홈 페이지](images/devhome_capture.png)   
  
**미디어 캡처** 페이지에는 다음과 같은 창이 있습니다.   
 
   *  [비디오 캡처](#ID4EHB)  

 
<a id="ID4EHB"></a>

   

## <a name="video-capture"></a>비디오 캡처  
   
  
**비디오 캡처** 창에는 개발자 콘솔의 비디오 스트림을 녹화하는 도구가 포함되어 있습니다. 또한 콘솔에서 마지막 몇 초 동안의 비디오를 캡처할 수 있습니다.
