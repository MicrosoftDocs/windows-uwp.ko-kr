# <a name="diffuse-light-maps"></a>확산 조명 맵


무광택 표면에 확산 조명 반사가 있습니다. 확산 조명의 밝기는 광원 으로부터의 거리와 표면 표준 및 광원 방향 벡터 간의 각도에 따라 달라 집니다. 텍스처 광원 지도는 복잡 한 확산 조명을 시뮬레이션할 수 있습니다.

조명 계산에 의해 시뮬레이션 된 확산 조명 효과는 일반적인 효과만 생성 합니다. 응용 프로그램은 텍스처 광원을 사용 하 여 더 복잡 한 확산 조명을 시뮬레이션할 수 있습니다.
