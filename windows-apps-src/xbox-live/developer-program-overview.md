# <a name="developer-program-overview"></a>개발자 프로그램 개요

Xbox Live를 사용 하도록 설정 제목 개발 하려는 경우 일부의 옵션이 있습니다를 사용할 수 있습니다. 각 다양 한 수준의 시간 투자 귀하를 사용할 수 있는 기능을 제공 하 고 옵션을 지원 합니다.

## <a name="xbox-live-creators-program"></a>Xbox Live 크리에이터스 프로그램

Xbox Live 크리에이터 스 프로그램은 개발 Xbox Live를 사용 하 여 잘 이해 하려는 경우 Xbox live는 좋은 시작 지점입니다. Microsoft에서 승인 프로세스가이 프로그램에 참여 하는 데 필요 하 고 최소한의 인증 및 게시 요구 사항입니다.

Xbox Live 크리에이터 스 프로그램의 제목 만들기를만 지원 합니다 [유니버설 Windows 플랫폼](https://msdn.microsoft.com/en-us/windows/uwp/get-started/universal-application-platform-guide) (UWP).  이러한 제목 UWP 게임을 Windows 10 Pc 및 Xbox One 콘솔 실행으로 생성 합니다.  Xbox One에서 UWP 게임을 실행 하는 방법에 대 한 자세한 내용은 참조 하세요. [Xbox One에 UWP](https://docs.microsoft.com/en-us/windows/uwp/xbox-apps/index)합니다.  

Xbox One에서 게이머 큐 레이트 매장 환경을 제공 하는 Xbox Live 크리에이터 스 프로그램을 통해 게시 된 게임 Xbox에서 Microsoft 스토어의 새 작성자 컬렉션 구역에서 판매 될 합니다. 이 누구 든 지 수를 개발 하 고 게임을 하 고 파악 하 고 예상 게이머 제대로 찾아 오셨습니다 큐 레이트 저장소 환경 콘솔을 제공 하는 개방형 플랫폼을 확인 간의 균형을 제공 합니다. Windows 10에서 제목 간의 모든 다른 Xbox Live 게임 Microsoft Store 게시 됩니다.

에 등록 해야 합니다 [파트너 센터 개발자 프로그램](https://developer.microsoft.com/store/register) 게임을 Xbox Live 크리에이터 스 프로그램의 일부로 릴리스 합니다. 두 가지 방법으로 게임을 따라야 하는 요구 사항:

1. Xbox Live에 로그인을 통합 하 고 (예: 게이머 태그, Gamerpic) 사용자 id를 표시 합니다. 다른 모든 Xbox Live 서비스는 선택적입니다.
2. 표준에 따릅니다 [Microsoft Store 정책](https://msdn.microsoft.com/en-us/library/windows/apps/dn764944.aspx)합니다.

## <a name="idxbox"></a>ID@Xbox

Xbox Live 크리에이터 스 프로그램 많은 게임 및 개발자에 적합합니다. 그러나 전체 Xbox Live 스택의 멀티 플레이어 온라인 게임, 성과, 게임 등을 액세스 하려는 또는 Xbox One 하드웨어 개발 키트를 사용 하 여 장치 제품군의 모든 기능에 액세스 하려는 경우는 [ ID@Xbox ](https://www.xbox.com/en-US/developers/id) 프로그램 사용해 보십시오입니다.

게임을 ID@Xbox 프로그램 개념 승인 이어야 하며 사용자의 큰 시간 약정은 Xbox One 및 Windows 10에서 전체 인증을 통해 이동 합니다.
ID@Xbox 제목 배치를 고객에 게 큰 노출 될 수 있는 작성자 컬렉션에 비해 저장소의 기본 섹션에서을 가져옵니다.

개발자는 ID@Xbox 프로그램에 액세스할 수도 개발자 지원 및 판촉 지원 Microsoft 뿐 아니라 개인 백서 및 개발자의 완전 한 보완 기술 포럼입니다. 계속 사용할 수 있습니다 [MSDN 포럼](https://social.msdn.microsoft.com/Forums/en-US/home?forum=xboxlivedev) 프로그래밍 관련된 질문에 게 문의 하거나 [Stack Overflow](https://stackoverflow.com/questions/tagged/xbox-live) "xbox live" 태그를 사용 하 여 원하는 경우.
