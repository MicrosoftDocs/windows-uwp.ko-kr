# <a name="serviceconfigsscidhoppershoppername"></a>/serviceconfigs/{scid}/hoppers/{hoppername}

일치 티켓을 만들려면 POST 작업을 지원 합니다.

> [!IMPORTANT]
> 이 URI 103 이상 계약과 함께 사용 하기 위한 하며 헤더 요소를 X Xbl-계약 버전: 103 또는 나중에 모든 요청 합니다.
