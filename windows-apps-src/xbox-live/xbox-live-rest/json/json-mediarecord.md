<a id="ID4EO"></a>

 
## <a name="mediarecord"></a>MediaRecord
 
MediaRecord 개체에 다음과 같이 지정 합니다.
 
| 멤버| 형식| 설명| 
| --- | --- | --- | 
| id| 문자열| Bing 카탈로그 또는 공급자 카탈로그에서 사용 하는 미디어의 식별자입니다.| 
| idType| 문자열| 미디어 식별자를 해석 하는 방법입니다. 가능한 값 "bing" 및 "provider"를 포함합니다.| 
| name| 문자열| 미디어 콘텐츠 지역화 된 이름입니다.|
