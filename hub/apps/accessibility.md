# <a name="accessibility-in-windows-10"></a>Windows 10의 내게 필요한 옵션

![hero-accessibility-bar-smaller](images/hero-accessibility-bar-smaller.png)

## <a name="build-accessibility-into-your-applications-to-empower-people-of-all-abilities"></a>모든 기능을 사용자에 게 제공할 수 있도록 응용 프로그램에 내게 필요한 옵션 빌드

전자 미디어를 비롯 한 제품 및 서비스는 가능한 한 많은 사용자를 위해 완벽 하 고 성공적인 환경을 제공 하도록 설계 된 경우에도 액세스할 수 있습니다.

장애가 있는 사용자 (임시 및 영구), 개인 기본 설정, 특정 작업 스타일 또는 상황 제약 조건 (예: 공유 작업 공간)에 대해 향상 된 기능 및 유용성으로 액세스 가능 하 고 포함 된 Windows 응용 프로그램을 빌드 하세요. 교통, 요리, 눈부심 등). 일부 일반적인 솔루션에는 대체 형식으로 정보 제공 (예: 비디오 캡션) 또는 보조 기술 (예: 화면 읽기) 사용 설정이 포함 됩니다.

**모든 사람이 계단 또는 엘리베이터를 사용 해야 하는지 여부에 관계 없이 건물에서 동일한 방에 액세스할 수 있어야 합니다.**

이 페이지에서는 다양 한 Windows 개발 프레임 워크에서 Windows 응용 프로그램을 빌드하는 개발자를 위한 내게 필요한 옵션 지원, 화면 판독기 및 돋보기와 같은 도구를 빌드하는 보조 기술 개발자를 위한 정보를 제공 합니다. 테스트 엔지니어가 응용 프로그램 테스트를 위한 자동화 된 스크립트를 만듭니다.
