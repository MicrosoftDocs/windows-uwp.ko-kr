# <a name="titlestitleidclusters"></a>/titles/{titleId}/clusters
제목에는 Xbox Live 계산 서버 인스턴스를 만들 수 있는 URI입니다. 이러한 Uri에 대 한 도메인이 `gameserverds.xboxlive.com` 고 `gameserverms.xboxlive.com`입니다.
 
  * [URI 매개 변수](#ID4EU)
  * [호스트 이름](#ID4EIB)
  * [올바른 메서드](#ID4EPB)
 
<a id="ID4EU"></a>

 
## <a name="uri-parameters"></a>URI 매개 변수
 
| 매개 변수| 설명| 
| --- | --- | 
| titleId| ID는 요청에서 작동 해야 하는 제목입니다.| 
  
<a id="ID4EIB"></a>

 
## <a name="host-name"></a>호스트 이름
 
gameserverms.xboxlive.com
  
<a id="ID4EPB"></a>

 
## <a name="valid-methods"></a>올바른 메서드
  
[POST](uri-titlestitleidclusters-post.md)
 
&nbsp;&nbsp;URI를 사용 하면 Xbox Live 계산 서버 인스턴스를 만들려면 클라이언트입니다.
