## <a name="requirement"></a>요구 사항
 
요구 사항은 개체에 다음과 같이 지정 합니다.
 
| 멤버| 형식| 설명| 
| --- | --- | --- | 
| id| 문자열| ID 요구 사항입니다.| 
| 현재| 문자열| 요구 사항까지 작업 진행의 현재 값입니다.| 
| target| 문자열| 요구 사항의 대상 값입니다.| 
| operationType| 문자열| 요구 사항 작업 유형입니다. 유효한 값은 합계, 최소, 최대입니다.| 
| ruleParticipationType| 문자열| 참여 형식 요구 사항입니다. 유효한 값은 개인, 그룹입니다.|
