# <a name="game-input-for-directx-games"></a>DirectX 게임의 게임 입력

이 섹션에서는 DirectX 게임을 위한 다양한 유형의 입력을 추가하는 방법에 대한 정보를 제공합니다.

게임 항목의 마우스 보기 컨트롤에서는 기존 마우스 및 키보드 컨트롤을 DirectX 게임에 추가하는 방법을 설명합니다.
