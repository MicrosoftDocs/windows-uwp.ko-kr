# <a name="authorization-types"></a>권한 부여 유형
 
Xbox Live 서비스는 요청을 사용 하 여 다양 한 인증 형식을 사용합니다. 이 항목에서는 표준 권한 부여 유형을 설명 합니다.
 
| 형식| 설명| 
| --- | --- | 
| 사용자 | 호출자에 게 권한이 부여 된 Xbox Live 사용자입니다. | 
| 제목 | 호출자가 신뢰할 수 있는 플랫폼의 제목입니다.|
