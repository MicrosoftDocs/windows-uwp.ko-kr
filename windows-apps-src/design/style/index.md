# <a name="style-for-uwp-apps"></a>UWP 앱의 스타일

스타일은 앱을 돋보이게 합니다. 핵심은 미적인 선택과 미묘하고 강력한 효과 사이에서 과감한 선택을 하는 것입니다. 핵심은 아름다움을 창조하는 것입니다. 

## <a name="style-and-the-fluent-design-system"></a>스타일과 흐름 디자인 시스템

<p>흐름 디자인 시스템을 사용하여 광원, 깊이, 움직임, 재료, 배율이 어우러지는 현대적이고 깔끔한 UI를 만들 수 있습니다. 이 스타일 지침은 색상, 입력 체계, 그리고 <a href="../style/acrylic.md">아크릴 재질</a> 및 <a href="../style/reveal.md">표시</a>와 같은 새로운 기능을 통해 흐름 디자인을 만드는 데 도움이 됩니다. 
</p>
