# <a name="performance-page-dev-home"></a>성능 페이지 (개발자 홈)
   
  
개발자 홈의 **성능** 페이지에는 다양 한 디스크 작업 및 문제해결을 위해 CPU 사용량 조건 시뮬레이션 합니다.   
 ![개발자 홈의 성능 페이지](images/devhome_performance.png)   
  
**성능** 페이지에는 다음과 같은 창을 포함 됩니다.   
 
   *  [디스크 시뮬레이션](#ID4EEB)  
   *  [CPU 시뮬레이션](#ID4EOB)  

 
<a id="ID4EEB"></a>

   

## <a name="disk-simulation"></a>디스크 시뮬레이션  
   
  
**디스크 시뮬레이션** 창 시스템 또는 다른 응용 프로그램에서 디스크 활동을 시뮬레이션합니다.   
  
<a id="ID4EOB"></a>

   

## <a name="cpu-simulation"></a>CPU 시뮬레이션  
   
  
**CPU 시뮬레이션** 창 CPU6에 대 한 작업을 시뮬레이션 (공유 코어).   
  
<a id="ID4EYB"></a>
