# <a name="apis-for-web-development"></a>웹 개발용 Api

이 목록은 웹 개발자가 관심을 가질 수 있는 다양 한 Microsoft의 Api를 포함 합니다. [Windows 10의 웹 개발 개요 페이지](/overview.md)에서 웹 개발에 집중 된 추가 도구와 서비스를 찾아보세요.

- **[Academic Knowledge API](https://azure.microsoft.com/services/cognitive-services/academic-knowledge/)**<br>
Academic Knowledge API를 사용 하 여 Microsoft Academic Graph의 다양 한 교육 콘텐츠를 활용 하세요.

- **[적응형 카드](https://adaptivecards.io/)**<br>
개발자가 실행 가능한 콘텐츠를 앱에 직접 삽입할 수 있도록 일반적인 일관 된 방식으로 UI 콘텐츠를 교환할 수 있도록 하는 개방형 카드 교환 형식입니다.

- **[ASP.NET 웹 Api](https://dotnet.microsoft.com/apps/aspnet/apis)**<br>
C #을 사용 하 여 모든 플랫폼에서 보안 REST Api를 빌드합니다.

- **[Azure DevOps Services REST API](/rest/api/azure/devops/)**<br>
Azure DevOps를 사용 하면 온-프레미스와 클라우드 모두에서 원본 제어, 작업 추적, 지속적인 통합 및 전달을 통해 소프트웨어를 개발할 수 있습니다.

- **[Azure Media Player](https://azure.microsoft.com/services/media-services/media-player/)**<br>
최적의 재생 형식을 자동으로 선택 하는 웹 친화적인 미디어 플레이어로, Flash 및 Silverlight에 대 한 대체 (fallback)를 포함 하 고, 통합 콘텐츠 보호를 제공 합니다.

- **[Bing Ads API](/advertising/guides/)**<br>
Bing Ads는 사용자의 검색 쿼리에서 사용 되는 키워드를 기준으로 광고를 표시 하는 데 사용 되는 APPC (클릭 별 지불) 광고 플랫폼입니다.

- **[Bing 기술 위젯](https://www.bing.com/widget/knowledge)**<br>
Bing 기술 자료 리포지토리의 기능을 활용 하 여 웹 사이트를 개선 하 고 페이지 컨텍스트 내에서 검색 된 엔터티에 대 한 관련 정보를 사용자에 게 제공 합니다.

- **[Bing 지도 REST 서비스 API](/bingmaps/rest-services/index.md)**<br>
지 오 코딩, 역방향 지 오 코딩, 라우팅 및 정적 이미지와 같은 작업을 수행 하기 위한 서비스입니다.

- **[Bing 지도 웹 컨트롤](https://www.bing.com/api/maps/sdkrelease/mapcontrol/isdk#overview)**<br>
비즈니스 인텔리전스 시나리오에 최적화 된 Microsoft의 최신 웹 매핑 개발자 키트입니다.

- **[Bing Search API](https://www.microsoft.com/bing/apis)**<br>
Bing Search API 웹 검색, 이미지 검색, 뉴스 검색, 사용자 지정 검색, autosuggest, 맞춤법 검사, 시각적 검색, 엔터티 검색 및 비디오 검색을 포함 합니다.

- **[Bing 공간 Data Services REST API](/bingmaps/spatial-data-services/)**<br>
Batch 지 오 코딩 (POI) 데이터 및 공간 데이터를 저장 하 고 노출 하는 기능입니다.

- **[Cognitive Services API](/azure/cognitive-services/)**<br>
사용자 요구 사항을 확인 하 고, 듣고, 말하고, 이해 하 고, 해석 하기 위해 지능적이 고 지원 되는 알고리즘을 앱, 웹 사이트 및 봇에 빌드 하세요. 이에 대 한 내용은 비전, 언어, 음성, 의사 결정, 검색, 인식 서비스 컨테이너 및 빅 데이터에 대 한 Cognitive Services입니다.

- **[Computer Vision API](/azure/cognitive-services/computer-vision/)**<br>
개발자에 게 이미지를 처리 하 고 정보를 반환 하기 위한 고급 알고리즘에 대 한 액세스를 제공 합니다.

- **[Content Moderator API](/azure/cognitive-services/content-moderator/overview.md)**<br>
텍스트, 이미지 및 비디오를 검색 하 고 콘텐츠 플래그를 자동으로 적용 하는 AI 기반 콘텐츠 조정 서비스는 사용자 검토자 팀을 위한 검토 도구인 온라인 중재자 환경입니다.

- **[얼굴 감지 API](/azure/cognitive-services/face/)**<br>
이미지에서 사람의 얼굴을 감지, 인식 및 분석 합니다.

- **[지식 탐색 서비스](https://github.com/microsoft/cognitive-research-technologies-docs/blob/master/knowledge-exploration-service/GettingStarted.md)**<br>
자연어 입력을 통해 구조적 데이터에 대한 대화형 검색 환경을 구현합니다.

- **[Language Understanding Intelligent Service (LUIS)](/azure/cognitive-services/luis/)**<br>
응용 프로그램을 사용 하 여 개인이 자신의 단어를 원하는 것을 이해할 수 있습니다.

- **[LinkedIn: 프로필에 추가](https://addtoprofile.linkedin.com/)**<br>
한 번의 클릭으로 구성원 프로필에 대 한 인증 및도를 가져옵니다.

- **[Microsoft 접합 API](/graph/use-the-api)**<br>
Microsoft Graph는 Microsoft 클라우드 서비스 리소스에 액세스할 수 있도록 하는 RESTful 웹 API입니다.

- **[모나코 편집기](https://microsoft.github.io/monaco-editor/)**<br>
모나코 편집기는 VS Code 및 F12를 지 원하는 코드 편집기입니다. 구문 색 지정, 풍부한 IntelliSense 및 diff 편집기에 대 한 지원을 포함 합니다.

- **[Microsoft 판매자 센터 콘텐츠 API](/advertising/shopping-content/)**<br>
Content API는 광고주가 Microsoft 판매자 센터 카탈로그를 프로그래밍 방식으로 관리할 수 있도록 하는 RESTful API입니다.

- **[Microsoft 팀 API](/graph/teams-concept-overview)**<br>
Microsoft Graph를 사용 하 여 채팅 기반 Microsoft 팀 작업 영역과 통합할 수 있습니다. 가상 팀을 만들어 비즈니스 문제를 해결 하 고, 적절 한 사용자를 추가 하 고, 채널, 탭 및 앱을 구성 하 고, 문제가 해결 되 면 팀을 보관 하거나 삭제 합니다.

- **[Personalizer Api](/azure/cognitive-services/personalizer/)**<br>
응용 프로그램에서 사용자에 게 표시할 최상의 환경을 선택 하 여 실시간 동작에서 학습할 수 있습니다.

- **[Power BI Embedded](https://azure.microsoft.com/en-us/services/power-bi-embedded/)**<br>
Power BI Embedded를 사용하면 Power BI 보고서를 웹 또는 모바일 애플리케이션에 통합할 수 있습니다.

- **[QnA Maker](/azure/cognitive-services/QnAMaker/)**<br>
반 구조화 된 콘텐츠에서 질문으로 이동 하 여 몇 분 안에 답변할 수 있습니다.

- **[Skype 웹 Api](/skype-sdk/websdk/docs/skypewebsdk)**<br>
실시간 공동 작업 모델을 비즈니스용 Skype 및 대규모 Skype 네트워크와 통합 하는 웹 환경을 빌드하기 위한 JavaScript Api 및 HTML 컨트롤의 집합입니다.

- **[음성 서비스 Api](/azure/cognitive-services/speech-service/)**<br>
음성을 인식하거나, 음성을 합성하거나, 실시간 번역을 받거나, 대화를 받아쓰거나, 음성을 봇 경험에 통합합니다.

- **[Text Analytics API](/azure/cognitive-services/text-analytics/)**<br>
원시 텍스트에 대 한 고급 자연어 처리를 제공 하며, 감정 분석, 핵심 문구 추출, 명명 된 엔터티 인식 및 언어 검색의 네 가지 주요 함수를 포함 합니다.

- **[통합 커뮤니케이션 웹 API 2.0](/skype-sdk/ucwa/unifiedcommunicationswebapi2_0)**<br>
    비즈니스용 Skype (인스턴트 메시징) 2015 IM (인스턴트 메시징) 및 현재 상태 기능을 제공 하는 REST API입니다.

- **[비디오 인덱서](/azure/media-services/video-indexer/)**<br>
Azure Media Player Video Indexer를 사용 하 여 비디오 및 오디오 모델을 통해 비디오에서 정보를 추출할 수 있습니다. 안정적인 비디오 출력을 생성 하 고, 동작을 감지 하 고, 지능적인 미리 보기를 만들고, 얼굴을 감지 하 고 추적 합니다.

- **[VS Code API](https://code.visualstudio.com/api/references/vscode-api)**<br>
Visual Studio Code 확장에서 호출할 수 있는 JavaScript Api 집합입니다. [확장 API](https://code.visualstudio.com/api) 를 사용 하면 기본 제공 되는 확장을 사용자 지정 하 고 VS Code 수 있습니다.

- **[Yammer REST API](https://developer.yammer.com/docs/getting-started)**<br>
Yammer를 사용 하 여 조직에서 커뮤니티 및 공동 작업을 빌드합니다. 플랫폼은 Yammer의 소셜 서비스를 응용 프로그램에 가져올 수 있도록 하는 개방형 Api 및 도구 집합으로 구성 됩니다.
