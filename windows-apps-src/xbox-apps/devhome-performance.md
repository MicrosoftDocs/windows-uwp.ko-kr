# <a name="performance-page-dev-home"></a>성능 페이지(개발자 홈)
   
  
개발자 홈의 **성능** 페이지에서는 문제 해결을 위해 다양한 디스크 작업 및 CPU 사용 조건을 시뮬레이션합니다.   
 ![성능 개발자 홈 페이지](images/devhome_performance.png)   
  
**성능** 페이지에는 다음과 같은 창이 있습니다.   
 
   *  [디스크 시뮬레이션](#ID4EEB)  
   *  [CPU 시뮬레이션](#ID4EOB)  

 
<a id="ID4EEB"></a>

   

## <a name="disk-simulation"></a>디스크 시뮬레이션  
   
  
**디스크 시뮬레이션** 창에서는 시스템 또는 다른 앱의 디스크 작업을 시뮬레이션합니다.   
  
<a id="ID4EOB"></a>

   

## <a name="cpu-simulation"></a>CPU 시뮬레이션  
   
  
**CPU 시뮬레이션** 창에서는 CPU6(공유 코어)의 작업을 시뮬레이션합니다.   
  
<a id="ID4EYB"></a>
