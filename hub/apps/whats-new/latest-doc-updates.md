# <a name="latest-updates-to-the-windows-developer-docs"></a>Windows 개발자 문서의 최신 업데이트

Windows 개발자 문서는 정기적으로 개선된 최신 정보와 콘텐츠로 업데이트됩니다. 다음은 2020년 9월 3일의 변경 내용에 대한 요약입니다.

참고: Windows 10 빌드 19041(2004라고도 함)의 일부로 추가된 특정 API 목록은 [이 목록](/windows/uwp/whats-new/windows-10-build-19041-api-diff)을 참조하세요.


## <a name="news"></a>뉴스

이번 달에는 탐색 속도를 높이고 오프라인 설명서 환경을 향상시키기 위해 수천 개의 절대 링크를 상대 링크로 변환했습니다. 끊어진 링크가 있으면 계속 알려주세요! 대부분의 페이지에서 피드백을 제공하거나 Twitter에서 **WindowsDocs** 핸들을 사용하여 연결할 수 있습니다.

이번 달의 주요 내용은 다음과 같습니다.

* WinUI [UI](https://docs.microsoft.com/windows/uwp/design/basics/xaml-basics-ui) 및 [데이터 바인딩](https://docs.microsoft.com/windows/uwp/data-binding/xaml-basics-data-binding), [샘플 코드](https://github.com/microsoft/Windows-appsample-photo-lab/tree/master/xaml-basics-starting-points)를 사용하는 4부로 구성된 자습서 시리즈에 대한 업데이트 

* [Windows 터미널 1.2](https://docs.microsoft.com/windows/terminal/) 및 미리 보기 1.3 설명서 업데이트

* 새로운 [개발 환경 설정](https://docs.microsoft.com/windows/dev-environment/overview) 페이지

* 새로운 몇 가지 [AI 및 기계 학습](https://docs.microsoft.com/windows/win32/direct3d12/gpu-faq) FAQ 업데이트

* [비동기 및 C++/WinRT와 C++/CX 간의 상호 운용성](https://docs.microsoft.com/windows/uwp/cpp-and-winrt-apis/interop-winrt-cx-async)에 대한 새로운 WinRT/C++ 항목
