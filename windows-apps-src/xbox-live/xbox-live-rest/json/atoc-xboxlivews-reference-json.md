# <a name="javascript-object-notation-json-object-reference"></a>JSON(JavaScript Object Notation) 개체 참조
 
개체 JSON (JavaScript Notation)는 웹에서 데이터를 캡슐화 하는 데는 경량, 표준을 기반으로 개체 지향 표기법입니다.
 
Xbox Live 서비스는 요청을 서비스의 응답에 사용 되는 JSON 개체를 정의 합니다. 이 섹션에서는 Xbox Live 서비스를 사용 하 여 사용 되는 각 JSON 개체에 대 한 참조 정보를 제공 합니다.
 
<a id="ID4EHB"></a>

[도전 과제 (JSON)](json-achievementv2.md)

&nbsp;&nbsp;도전 과제 개체 (버전 2)입니다.

[ActivityRecord (JSON)](json-activityrecord.md)

&nbsp;&nbsp;하나 이상의 사용자 다기능 현재 상태에 대 한 서식이 지정 되 고 지역화 된 문자열입니다.

[ActivityRequest (JSON)](json-activityrequest.md)

[GameClipUri (JSON)](json-gameclipuri.md)

[PermissionCheckResponse (JSON)](json-permissioncheckresponse.md)

&nbsp;&nbsp;단일 대상 사용자에 대 한 단일 사용 권한 설정에 대 한 단일 사용자 확인의 결과입니다.

[SessionEntry (JSON)](json-sessionentry.md)
