# <a name="point-of-service"></a>서비스 지점

| 항목 | 설명 |
|------|------------|
| [기본 사항](pos-basics.md) | 이 문서는 서비스 지점 UWP API를 시작하는 데 도움이 됩니다. |
| [하드웨어 지원](pos-device-support.md) | [Windows. PointOfService](https://docs.microsoft.com/uwp/api/Windows.Devices.PointOfService) UWP api와 호환 되는 알려진 장치 목록입니다. |
| [바코드 스캐너](pos-barcodescanner.md) | 바코드 스캐너에 대 한 아티클의 인덱스입니다. |
| [자기 줄무늬 판독기](pos-magnetic-stripe-reader.md) | 자기 줄무늬 판독기에 대 한 아티클의 인덱스입니다.
| [POSPrinter](pos-printer.md) | POSPrinter에 대 한 아티클의 인덱스입니다. |
