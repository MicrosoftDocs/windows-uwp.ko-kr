# <a name="fundamentals-of-directx-programming"></a>DirectX 프로그래밍의 기본 사항

이 섹션에서는 DirectX 프로그래밍의 기본 블록에 대 한 정보를 제공 합니다.

DirectX 게임을 위한 2D 그래픽 항목은 2D 비트맵 그래픽 및 효과의 사용 방법과 Direct2D 및 Direct3D 라이브러리의 요소 조합을 사용 하 여 게임에서 사용 하는 방법을 설명 합니다.

Direct3d 그래픽 학습 가이드 항목에서 사용 하는 개념에 대 한 개요를 제공 합니다.

DirectX 게임의 기본 3D 그래픽 항목에서는 Direct3D 개념 및 API를 소개 하는 다섯 부분으로 된 자습서를 사용 하는 3D 그래픽의 기본 개념을 설명 합니다. 이 자습서에서는 Windows 런타임 사용 하 여 Direct3D 인터페이스를 초기화 하 고, 꼭 짓 점 셰이더 작업을 적용 하 고, geometry를 설정 하 고, 장면을 래스터화 하 고, 숨겨진 표면을 추려내 하는 방법을 보여 줍니다.

DirectX 게임의 리소스 로드 항목에서는 UWP 게임에서 사용 하기 위해 로컬 저장소 또는 기타 데이터 스트림에서 메시 (모델), 질감 (비트맵) 및 컴파일된 셰이더 개체를 로드 하는 기본 단계를 안내 합니다.
