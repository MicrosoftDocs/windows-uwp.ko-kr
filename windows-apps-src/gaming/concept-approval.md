# <a name="concept-approval"></a>개념 승인

Xbox에서 실행되는 게임을 만들기 시작하면 게시하기 전에 먼저 *개념 승인*을 위해 해당 게임에 대한 제안을 Microsoft에 제출해야 합니다. 이 개괄적인 사전 제출은 게임에 대한 전체 계획에서 발생할 수 있는 어려움이나 단점을 프로세스의 맨 처음에 식별하여 Microsoft와 개발자 모두에게 이점을 제공합니다. 콘텐츠가 지나치게 상스럽거나, 공격적이거나 불쾌하지 않은지, 또한 대상 플랫폼에 편안하게 맞는지 확인합니다. 제안이 제출되면 Microsoft에서 이를 검토한 다음 결과를 알려 줍니다.

UWP(유니버설 Windows 플랫폼) 게임을 개발하는 경우 고객이 Xbox 디바이스에서 게임을 다운로드할 수 있도록 하거나 Xbox Live 기능을 사용하도록 설정하려면 개념 승인만 있으면 됩니다. UWP 게임을 Windows 데스크톱 또는 모바일 디바이스에서만 사용할 수 있도록 하려는 경우(또는 모든 디바이스를 대상으로 한, 게임이 *아닌* UWP 앱을 게시하는 경우) Xbox Live를 사용하고 있지 않으면 [개발자 계정](https://go.microsoft.com/fwlink/?LinkId=817223)만 있으면 되고 대시보드를 통해 자유롭게 앱을 구성하여 스토어에 제출할 수 있습니다.

## <a name="submit-your-concept-for-approval"></a>승인을 위해 개념 제출

독립 게임 개발자나 게시자인 경우 ID@Xbox 프로그램을 통해 개념을 제출하여 승인을 요청할 수 있습니다. ID@Xbox에 대해 자세히 알아보고 [여기](http://www.xbox.com/Developers/id)에서 신청하세요.

이미 ID@Xbox 개발자인 경우 게임 개념을 제출할 수 있는 GIF(게임 정보 양식)에 대한 링크를 전송 받았어야 합니다. 질문이 있는 경우 [id@xbox.com](mailto:id@xbox.com)에 문의하세요.

Microsoft와의 기존 사용권 계약이 있는 경우 개념 제출에 대한 내용은 Microsoft 계정 팀에 문의합니다.
