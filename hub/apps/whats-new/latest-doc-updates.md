# <a name="latest-updates-to-the-windows-developer-docs"></a>Windows 개발자 문서의 최신 업데이트

Windows 개발자 문서는 정기적으로 개선된 최신 정보와 콘텐츠로 업데이트됩니다. 2021년 2월 10일의 변경 내용은 다음과 같습니다.

참고: Windows 10 빌드 19041(2004라고도 함)의 일부로 추가된 특정 API 목록은 [이 목록](/windows/uwp/whats-new/windows-10-build-19041-api-diff)을 참조하세요.

최신 Windows 개발자 설명서 뉴스를 보거나 의견 및 질문을 남길 수 있는 Twitter 핸들은 [@WindowsDocs](https://twitter.com/windowsdocs)입니다.

저희는 모든 설명서에서 사려 깊지 못한 언어 사용을 지속적으로 찾아서 제거하고 있습니다.

Microsoft 문서에 기여하는 방법에 대한 자세한 내용은 [기여자 가이드](/contribute/)를 참조하세요.

이번 달의 주요 내용은 다음과 같습니다.

### <a name="new-content"></a>새로운 내용


* [C++/WinRT에서 C#으로 포팅](/windows/uwp/cpp-and-winrt-apis/move-to-winrt-from-csharp)
* [연습: C#/WinRT 구성 요소를 만들고 C++/WinRT에서 사용](/windows/uwp/csharp-winrt/create-windows-runtime-component-cswinrt)
