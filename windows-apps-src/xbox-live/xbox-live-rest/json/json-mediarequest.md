## <a name="mediarequest"></a>MediaRequest
 
MediaRequest 개체에 다음과 같이 지정 합니다.
 
| 멤버| 형식| 설명| 
| --- | --- | --- | 
| id| 문자열| Bing 카탈로그 또는 공급자 카탈로그에서 사용 하는 미디어의 식별자입니다.| 
| idType| 문자열| 해석 하는 방법 <b>id</b>합니다. 가능한 값 "bing" 및 "provider"를 포함합니다.|
