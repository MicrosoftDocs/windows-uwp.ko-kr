# <a name="microsoft-store-marketing-guidelines-for-apps"></a>Microsoft Store 앱에 대 한 지침을 마케팅

앱 및 Microsoft Store 콘텐츠를 홍보 하는 방법에 알아봅니다. 이러한 지침에서는 사용 가능한 자산을 사용하는 방법과 인쇄물, TV, 소셜 미디어 및 디지털 광고에서 앱을 홍보하는 것에 대한 권장 사항을 설명합니다.

## <a name="store-badges"></a>스토어 배지

Microsoft Store 앱의 목록에 더 많은 고객을 생성 하는 데 특별 한 프로 모션 배지를 만들었습니다. 이러한 배지는 44개 언어로 제공됩니다. 이러한 이미지를 사용할 경우 따라야 할 특정한 요구 사항이 있다는 사실에 주의하세요. 이러한 요구 사항은 사용 예제 및 지침과 함께 PDF 파일로 제공됩니다.

[![다운로드 단추](images/downloadbutton.png)](https://go.microsoft.com/fwlink/p/?LinkId=529769) **Microsoft Store 배지 지침 (PDF, 영어)**


## <a name="badge-generator-and-images"></a>배지 생성기 및 이미지

[배지 생성기](https://go.microsoft.com/fwlink/p/?LinkID=534236)를 사용하여 원하는 언어로 스토어 배지를 표시하고 앱의 스토어 목록으로 바로 연결되는 HTML을 생성할 수 있습니다.

아래 링크에서 PNG 및 PDF 형식의 배지 이미지 전체 집합을 다운로드할 수도 있습니다.

[![다운로드 단추](images/downloadbutton.png)](https://go.microsoft.com/fwlink/p/?LinkId=529771) **모든 배지 이미지**


## <a name="device-images"></a>장치 이미지

앱을 홍보하기 위해 Windows 장치에서 실행할 경우 앱이 어떻게 표시되는지를 보여 주고 싶을 수 있습니다. 휴대폰, 태블릿, 노트북, PC 등 선택할 수 있는 다양한 장치 섀시 이미지가 마련되어 있습니다. 이러한 이미지를 사용 하는 방법에 대 한 기술 사양 및 모범 사례에서 찾을 수 있습니다 합니다 [Microsoft Store 마케팅 지침](https://go.microsoft.com/fwlink/p/?LinkId=529769)합니다.

[![다운로드 단추](images/downloadbutton.png)](https://go.microsoft.com/fwlink/p/?LinkId=533057) **Windows 장치 아트**

## <a name="license-to-microsoft-marks"></a>Microsoft 마크에 사용권 부여

*Microsoft 표시* 것을 의미 합니다 *Microsoft 배지* 에 설명 된 합니다 [배지 생성기](https://go.microsoft.com/fwlink/p/?LinkID=534236) 페이지입니다. 이러한 배지를 사용하려면 다음과 같이 해야 합니다.

-   Microsoft Store 앱 또는 기타 콘텐츠를 사용할 수 있거나에 속해야 합니다 [Microsoft 제휴 프로그램](https://go.microsoft.com/fwlink/p/?LinkId=624463)합니다.

-   파트너 센터에서 앱 개발자로 등록 된 경우 준수 합니다 [라이선스를 Microsoft 표시 "를"](https://docs.microsoft.com/legal/windows/agreements/app-developer-agreement#license_to_mark) 앱 개발자 계약의 섹션입니다.

-   파트너 센터에서 앱 개발자로 등록 되지 않습니다 하는 경우 Microsoft는 배지를 사용 하 여 Microsoft Store 배지 지침에 설명 된 대로에 전 세계, 비독점적, 양도, 무료 라이선스를 부여 합니다. Microsoft는 이 지침을 변경할 수 있으며, 이 경우 Microsoft는 적절한 방법을 통해 규정이 게시된 새 URL로 리디렉션을 제공합니다. Microsoft는 여기에 명시적으로 부여되지 않은 모든 권한을 보유합니다.

-   로고 사용 사양에 설명 된 다음 합니다 [Microsoft Store 배지 지침](https://go.microsoft.com/fwlink/p/?LinkId=529769)합니다.

Microsoft는 Microsoft 마크 및 관련 영업권의 독점 소유자이며 Microsoft 마크 사용 관련 영업권의 독점 수혜자입니다. Microsoft는 언제든지 재량에 따라 이 라이선스를 해지할 수 있습니다.
