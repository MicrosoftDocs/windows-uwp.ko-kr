# <a name="settings-page-dev-home"></a>설정 페이지 (개발자 홈)
   
  
개발 홈의 **설정** 페이지의 개발 콘솔에 대 한 다양 한 설정에 대 한 액세스를 제공합니다.   
 ![설정 페이지의 개발 홈](images/devhome_settings.png)   
  
**설정** 페이지에는 다음 창을 포함 됩니다.   
 
   *  [콘솔 설정](#ID4EEB)  
   *  [운영 체제 복구 설정 업데이트](#ID4EOB)  
   *  [Visual Studio](#ID4EYB)  
   *  [디버그 설정](#ID4ECC)  
   *  [무인 설치 스크립트 구성](#ID4EMC)  
   *  [기본 홈 환경](#ID4E3C)  

 
<a id="ID4EEB"></a>

   

## <a name="console-settings"></a>콘솔 설정  
   
  
**콘솔 설정** 창에서 콘솔 설정 및 정보에 대 한 "한눈" 보기를 제공합니다. 이러한 도구를 사용 하 여를 보고 하는 콘솔의 호스트 이름, 샌드박스 및 보안 액세스 키 설정을 편집할 수 있습니다.   
  
<a id="ID4EOB"></a>

   

## <a name="os-recovery-update-settings"></a>운영 체제 복구 설정 업데이트  
   
  
**설정을 업데이트 하는 운영 체제 복구** 창에는 콘솔의 자동 업데이트 설정을 구성합니다. 이러한 설정을 변경 하는 참고 사항을 적용 하려면 24 시간까지 걸릴 수 있습니다.   
  
<a id="ID4EYB"></a>

   

## <a name="visual-studio"></a>Visual Studio  
   
  
**Visual Studio** 창에서 Visual Studio를 사용한 콘솔 한 쌍으로 사용 됩니다. 모든 쌍으로 지정 된 Visual Studio 인스턴스를 제거할 수도 있습니다.   
  
<a id="ID4ECC"></a>

   

## <a name="debug-settings"></a>디버그 설정  
   
  
**디버그 설정** 창을 보고 하 여 일반적인 전환 되는 디버그 설정 합니다.   
  
<a id="ID4EMC"></a>

   

## <a name="unattended-script-configuration"></a>무인 설치 스크립트 구성  
   
  
**무인 설치 스크립트 구성** 창 Xbox 하나의 콘솔에 대 한 설치 스크립트를 관리합니다.   
  
<a id="ID4E3C"></a>

   

## <a name="default-home-experience"></a>기본 홈 환경  
   
  
**홈 경험 기본** 창에서 콘솔 시작 하는 경우 소매 홈 화면이 나 [콘솔 (개발자 홈)에 대 한 개발자 홈](dev-home.md) 화면을 표시할지 여부를 결정 합니다.   
  
<a id="ID4EJD"></a>
