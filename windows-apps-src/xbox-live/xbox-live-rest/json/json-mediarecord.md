<a id="ID4EO"></a>

 
## <a name="mediarecord"></a>MediaRecord
 
MediaRecord 개체에는 다음 사양을 있습니다.
 
| 멤버| 유형| 설명| 
| --- | --- | --- | 
| id| string| Bing 카탈로그 또는 공급자 카탈로그를 사용 하는 미디어의 식별자입니다.| 
| idType| string| 미디어 식별자를 해석 하는 방법입니다. 가능한 값 "bing" 및 "공급자" 포함 됩니다.| 
| name| 문자열| 미디어 콘텐츠의 지역화 된 이름입니다.|
