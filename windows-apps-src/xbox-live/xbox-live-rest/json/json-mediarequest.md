## <a name="mediarequest"></a>MediaRequest
 
MediaRequest 개체에는 다음 사양을 있습니다.
 
| 멤버| 유형| 설명| 
| --- | --- | --- | 
| id| string| Bing 카탈로그 또는 공급자 카탈로그에서 사용 되는 미디어의 식별자입니다.| 
| idType| string| <b>Id</b>를 해석 하는 방법입니다. 가능한 값에는 "bing" 및 "공급자" 포함 됩니다.|
