# <a name="getting-started-with-uwp-for-ios-developers"></a>iOS 개발자용 UWP 시작


인 경우 Windows 10에 대 한 개발 고려 iOS 개발자는 이러한 문서를 시작 하기에 적합 합니다. 앱 작성을 시작 하면을 해야 하는 개념 중 일부에 대해 알아봅니다는 및 Microsoft Store 작업을 게시 하는 방법을 다룹니다.

이 섹션에서는 점진적으로 시작하면서 Microsoft Visual Studio 및 C# 프로그래밍 언어를 사용하여 간단한 앱을 만드는 방법을 살펴보고 특히 현재 사용하는 도구에 따라 프로세스가 어떻게 달라지는지에 대해 설명합니다. C#을 잘 몰라도 됩니다. 걱정 하지 마십시오 다른 프로그래밍 언어와 도구를 사용할 수 있고이 대 한 [시작 합니다. 프로그래밍 언어 선택](getting-started-choosing-a-programming-language.md)).

Windows 10 데스크톱, 랩톱, 태블릿, phone 장치에서 매력적인 앱을 만들기 위한 새 플랫폼을 소개 합니다. UWP(유니버설 Windows 플랫폼) 앱은 많은 고유한 기능을 제공하므로 iOS 앱을 직접 포팅하면 이러한 기능을 사용하지 못할 수 있습니다. 따라서 새 컨트롤 및 기능을 사용하여 개발자로서 생활을 더 편리하게 하고 새 앱을 가능하게 만드는 방법을 확인하는 것이 좋습니다.

앱을 포팅하지만 말고 **reimagine**하여 새로운 기능 및 새 장치를 활용하는 것이 좋습니다. 최소 공통 분모에 대 한 납부 하지 않습니다 하지만 Cortana를 사용 하 여 라이브 타일, 알림 및 상호 작용 등의 고유한 Windows 10 기능을 사용 하는 다양 한 환경을 만듭니다.

이러한 연습을 사용 하 여 시작 하려면 Windows 10 및 Microsoft Visual Studio가 설치 된 컴퓨터를 해야 합니다. 다운로드할 수 있습니다 합니다 [UWP 앱 프로그래밍을 위한 개발자 다운로드](https://developer.microsoft.com/en-us/windows/downloads)합니다. PC가 없는 경우 걱정하지 마세요. Mac을 사용할 수 있습니다. [Mac에 Windows 및 개발 도구 설치](setting-up-your-mac-with-windows-10.md)를 참조하세요.
