# <a name="configure-access-policies-in-partner-center"></a>파트너 센터에서 액세스 정책 구성

사용할 수 있습니다 [파트너 센터](https://partner.microsoft.com/dashboard) 다른 서비스, 게임 및 앱 타이틀의 Xbox Live 설정 및 데이터에 액세스를 허용 하도록 합니다. 예를 들어, 웹 사이트에서 순위표를 표시 하는 웹 서비스를 확인할 수 있습니다 하거나 보거나 저장 된 게임 데이터를 수정 하려면 게임의 제목 저장소에 액세스할 수 있는 도우미 앱을 할 수 있습니다.

기본적으로 자체 제목 필드만 설정 및 Xbox Live 서비스에 저장 된 데이터를 액세스할 수 있습니다. 파트너 센터에서 액세스 정책을 구성 하 여이 변경할 수 있습니다.

> [!NOTE]
> 이 항목에서는 Xbox Live 크리에이터 스 프로그램 제목에 적용 되지 않습니다.

다음을 수행 하 여 구성을 추가 합니다.

1. 제목을 선택한 후 [파트너 센터](https://partner.microsoft.com/dashboard), 이동할 **Services** > **Xbox Live**합니다.

2. 링크를 클릭 **액세스 정책**합니다.

3. 액세스 권한을 부여 하려는 설정을 클릭 하 고 추가 앱/서비스 단추를 클릭 합니다. 이렇게 하면 해당 설정에 액세스 하도록 구성 된 앱/서비스의 목록 맨 아래에 새 행을 추가 됩니다.

4. 드롭다운 상자에서 앱 또는 서비스의 유형을 선택 하 고 앱, 제목 id 또는 앱 또는 데이터를 액세스 하는 서비스의 서비스 id를 나타내기 위해 세부 정보 상자를 채웁니다.

5. 앱 또는 서비스에 데이터를 읽기만 가능 또는 전체 데이터에 액세스할 수 있는 경우 선택 합니다.

6. 각 앱 또는 해당 설정에 액세스 해야 하는 서비스 및 각 설정에 대해 반복 합니다. 클릭할 수 **삭제** 엔트리를 제거 하려고 합니다.

7. 완료 되 면 클릭 합니다 **저장** 변경 내용을 저장 하는 단추입니다.
