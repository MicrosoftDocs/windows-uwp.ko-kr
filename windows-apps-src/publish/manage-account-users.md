# <a name="manage-account-users"></a>계정 사용자 관리

Azure Active Directory를 사용 하 여 추가 하 고 추가 사용자에 게 관리 하 [파트너 센터](https://partner.microsoft.com/dashboard) 계정. 각 사용자가 갖는 역할 또는 사용자 지정 권한을 정의할 수 있습니다. 또한 사용자 그룹 또는 Azure AD 응용 프로그램에 역할을 할당할 수 있습니다.

추가 하 고 계정 사용자 관리를 위해 조직의 Azure Active Directory를 사용 하 여 파트너 센터 계정을 먼저 연결 해야 합니다. 

이 섹션에는 다음을 수행하는 방법이 설명되어 있습니다.

-   [파트너 센터 계정을 사용 하 여 Azure Active Directory 연결](associate-azure-ad-with-dev-center.md)
-   [사용자, 그룹 및 파트너 센터 계정에 Azure AD 응용 프로그램 추가](add-users-groups-and-azure-ad-applications.md)
-   [역할 및 계정 사용자에 대 한 사용자 지정 권한 설정](set-custom-permissions-for-account-users.md)

> [!TIP]
> 이러한 항목은 파트너 센터에서 Windows 앱 개발자 프로그램에 특정 테 넌 트를 연결 및 사용자 관리 Windows 하드웨어 개발자 프로그램에서 계정에 대 한 마찬가지로 작동 하지만 (참조 [대시보드 관리](https://docs.microsoft.com/windows-hardware/drivers/dashboard/dashboard-administration) 에 대 한 자세한 정보) 또는 Windows 데스크톱 응용 프로그램 (참조 [Windows 데스크톱 응용 프로그램](https://docs.microsoft.com/windows/desktop/appxpkg/windows-desktop-application-program#add-and-manage-account-users) 자세한 정보에 대 한).
