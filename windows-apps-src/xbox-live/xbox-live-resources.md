# <a name="xbox-live-resources"></a>Xbox Live 리소스

이 항목에서는 Xbox Live 게임을 개발 하기 위한 유용한 리소스에 대 한 링크를 포함 합니다.

## <a name="api-reference"></a>API 참조

WinRT SDK API 참조를 확인할 수 있습니다 온라인:

* [Xbox Live WinRT SDK](https://docs.microsoft.com/en-us/dotnet/api/?view=xboxlive-dotnet-2017.11.20171204.01)

C + + SDK는 현재 오프 라인 형식으로 사용할 수만 있습니다.

전체 API 참조 설명서를 포함 하는 WinRT 뿐만 아니라 c + +에서 다운로드할 수 있습니다.

* [Xbox Live API 참조](https://aka.ms/xboxliveuwpdocs)

REST API 정보를 찾을 수 있습니다.

* [Xbox Live RESTful 참조](xbox-live-rest/atoc-xboxlivews-reference.md)


## <a name="xbox-live-api-source-code"></a>Xbox Live API 소스 코드

Xbox Live API 소스 코드를 GitHub의에서 공개적으로 제공 됩니다.

* [Xbox Live API 원본](https://github.com/Microsoft/xbox-live-api)

## <a name="xbox-live-unity-plug-in"></a>Xbox Live Unity 플러그 인

Xbox Live 크리에이터 스 프로그램에서 사용 하 여 Unity 게임을 개발 하는 개발자에 게는 Xbox Live Unity 플러그 인에서 GitHub에서 제공 됩니다.

* [Xbox Live Unity 플러그 인](https://github.com/Microsoft/xbox-live-unity-plugin)

> [!WARNING]
> Xbox Live 크리에이터 스 프로그램 도구 이기 때문에 Xbox Live Unity 플러그 인에 제품군 Xbox Live 기능을 제공 하지 않습니다. 참조를 [개발자 프로그램 개요 기능 테이블](developer-program-overview.md#feature-table) 보려면 어떤 Xbox Live 서비스를 사용할 수 있습니다.

## <a name="xbox-live-platform-extensions-sdk"></a>Xbox Live SDK 플랫폼 확장

Winmd 파일을 포함 하는이 패키지 및 설명서와 관련 된 네트워킹 보안 및 연결 된 저장소의 UWP 플랫폼에 대 한 기능입니다. 이 확장 Windows.Gaming.XboxLive.Storage 네임 스페이스를 사용 하 여 UWP 제목에 연결 된 저장소의 사용을 사용 하도록 설정 됩니다.

* [Xbox Live SDK 플랫폼 확장](https://aka.ms/xblextsdk)

> [!NOTE]
> - 개발자 환경에서 Windows 10 Fall Creators Update SDK를 사용 하는 경우에이 다운로드는 필요 하지 이상.
> - 보안 소켓, Xbox Live 크리에이터 스 프로그램의 개발자에 게 제공 되지 않습니다.

## <a name="xbox-live-managed-partner-only-content"></a>Xbox Live 관리 되는 파트너는 콘텐츠

Xbox Live에 대 한 일부 문서는 관리 되는 파트너 에게만 제공 됩니다. 이 정보에 액세스 하려면 Microsoft 계정 (MSA)가 Xbox Live에 완전히 액세스 가능한 있어야 합니다.

권한이 부여 된 MSA에 있는 경우 찾을 수 있습니다 파트너만 콘텐츠 [Xbox Live 파트너 콘텐츠](https://developer.microsoft.com/en-us/games/xbox/docs/xboxlive/xbox-live-partners/partner-content)합니다.
