# <a name="3d-printing"></a>3D 인쇄


이 섹션에서는 유니버설 Windows 앱에 3 차원 인쇄 기능을 추가 하려면 [Windows.Graphics.Printing3D 네임 스페이스](https://msdn.microsoft.com/library/windows/apps/windows.graphics.printing3d.aspx) 를 활용 하는 방법을 설명 합니다.  

에 대 한 자세한 내용은 Windows 10, 하드웨어 파트너, 커뮤니티 토론 포럼 및 3 차원 인쇄 기능에 대 한 일반적인 정보에 대 한 리소스를 포함 하 여 3 차원 인쇄 하드웨어 개발자 센터에서 사이트를 [3 차원 Windows 10을 사용 하 여 인쇄](https://developer.microsoft.com/windows/hardware/3d-print-support-windows-10) 를 참조 하십시오.

| 항목 | 설명 |
|-------|-------------|
| [앱에서 3D 인쇄](3d-print-from-app.md) | UWP 앱에 3 차원 인쇄 기능을 추가 하는 방법에 알아봅니다. 이 항목에서는 3D 모델이 인쇄 가능하고 올바른 형식인지 확인한 다음 3D 인쇄 대화 상자를 시작하는 방법을 설명합니다. |
| [3MF 패키지 생성](generate-3mf.md) | 3 차원 제조 서식 파일 유형 및 방법을 것 생성 및 조작할 수 Windows.Graphics.Printing3D Api와의 구조에 설명 합니다. |
