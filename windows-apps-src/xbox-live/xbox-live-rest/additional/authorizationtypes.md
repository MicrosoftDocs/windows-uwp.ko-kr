# <a name="authorization-types"></a>권한 부여 유형
 
Xbox Live 서비스 요청을 사용 하 여 다양 한 권한 부여 유형 사용합니다. 이 항목에서는 표준 권한 부여 유형 설명 합니다.
 
| 유형| 설명| 
| --- | --- | 
| 사용자 | 호출자는 권한 있는 Xbox Live 사용자 합니다. | 
| 제목 | 호출자는 신뢰할 수 있는 플랫폼에서 제목입니다.|
