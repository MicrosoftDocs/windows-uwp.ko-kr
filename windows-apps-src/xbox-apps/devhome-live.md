# <a name="xbox-live-page-dev-home"></a>Xbox Live 페이지 (개발자 홈)
   
  
개발자 홈의 **Xbox Live** 페이지 멀티 플레이 정보를 캡처하고 Xbox Live 서비스의 현재 상태를 표시 합니다.   
 ![개발자 홈의 Xbox Live 페이지](images/devhome_live.png)   
  
**Xbox Live** 페이지는 다음과 같은 창을 포함 되어 있습니다.   
 
   *  [XBTrace (advancedmp)](#ID4EPB)  
   *  [Xbox Live 상태](#ID4E3B)  

 
<a id="ID4EPB"></a>

   

## <a name="xbtrace-advancedmp"></a>XBTrace (advancedmp)  
   
  
**XBTrace (advancedmp)** 페이지를 시작 하거나 멀티 플레이어 추적 캡처를 중지 합니다. 또한 현재 멀티 플레이 상태의 스냅숏을 캡처할 수 있습니다.   
  
<a id="ID4E3B"></a>

   

## <a name="xbox-live-status"></a>Xbox Live 상태  
   
  
**Xbox Live 상태** 창에서 Xbox Live 서비스 문제 또는 중단에 대 한 피드 상태를 표시 합니다.   
  
<a id="ID4EPC"></a>
