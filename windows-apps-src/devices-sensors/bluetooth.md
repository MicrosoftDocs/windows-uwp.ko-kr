# <a name="bluetooth"></a>Bluetooth
이 단원에서는 Bluetooth 범용 Windows 플랫폼 (UWP) 응용 프로그램에 통합 하는 방법에 대 한 문서. 응용 프로그램을 구현 하도록 선택할 수 있습니다 하는 두 가지 다른 bluetooth 기술이 있습니다.

## <a name="classic-bluetooth-rfcomm"></a>클래식 Bluetooth (RFCOMM)
Bluetooth LE 이전 장치 Bluetooth를 사용 하 여 통신이 프로토콜 일반적으로 사용 했습니다. 이 프로토콜에는 간단 하 고 에너지 절약의 필요 없이 장치를 통신 하는 데 유용 합니다. 코드 샘플을 포함 하 여이 프로토콜에 대 한 자세한 내용은 [Bluetooth RFCOMM](send-or-receive-files-with-rfcomm.md) 항목을 참조 하십시오.

## <a name="bluetooth-low-energy-le"></a>Bluetooth 낮은 에너지 (LE)
Bluetooth 낮은 에너지 (LE) 검색과 효율적인 에너지 사용량 필요 없는 장치 간의 통신을 위한 프로토콜을 정의 하는 사양입니다. 자세한 내용은 코드 샘플과 함께 [낮은 에너지 Bluetooth](bluetooth-low-energy-overview.md) 항목을 참조.
