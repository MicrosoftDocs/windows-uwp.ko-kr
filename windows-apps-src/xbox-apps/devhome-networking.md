# <a name="networking-page-dev-home"></a>네트워킹 페이지 (개발자 홈)
   
  
개발자 홈의 **네트워킹** 페이지 문제해결을 위해 다양 한 네트워킹 조건 유사한 기능을 제공 합니다. 또한 네트워크 패킷 추적을 시작할 수 있습니다.   
 ![개발자 홈의 네트워킹 페이지](images/devhome_networking.png)   
  
**네트워킹** 페이지에는 다음과 같은 창을 포함 됩니다.   
 
   *  [네트워크 시뮬레이션](#ID4EEB)  
   *  [XBTrace (netcap)](#ID4EOB)  

 
<a id="ID4EEB"></a>

   

## <a name="network-simulation"></a>네트워크 시뮬레이션  
   
  
**네트워크 시뮬레이션** 창에서 다양 한 대역폭 프로필에 대 한 테스트 네트워크 시뮬레이션을 실행 합니다.   
  
<a id="ID4EOB"></a>

   

## <a name="xbtrace-netcap"></a>XBTrace (netcap)  
   
  
**XBTrace (netcap)** 창을 시작 하거나 네트워크 패킷 추적을 중지 합니다.   
  
<a id="ID4E2B"></a>
