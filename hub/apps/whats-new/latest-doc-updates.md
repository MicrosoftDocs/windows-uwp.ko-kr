# <a name="latest-updates-to-the-windows-developer-docs"></a>Windows 개발자 문서의 최신 업데이트

Windows 개발자 문서는 정기적으로 개선된 최신 정보와 콘텐츠로 업데이트됩니다. 2020년 10월 5일 현재의 변경 내용에 대한 요약은 다음과 같습니다.

참고: Windows 10 빌드 19041(2004라고도 함)의 일부로 추가된 특정 API 목록은 [이 목록](/windows/uwp/whats-new/windows-10-build-19041-api-diff)을 참조하세요.


## <a name="news"></a>뉴스


이번 달에는 탐색 속도를 높이고 오프라인 읽기 환경을 향상시키기 위해 일단의 또 다른 절대 링크를 상대 링크로 변환했습니다. 끊어진 링크가 있으면 계속 알려주세요. 대부분의 페이지에서 직접 피드백을 제공하거나 Twitter에서 [@WindowsDocs](https://www.twitter.com/windowsdocs) 핸들을 사용하여 연락할 수 있습니다.

이번 달의 주요 내용은 다음과 같습니다.

* [Surface Duo 설명서](https://docs.microsoft.com/dual-screen/introduction)가 업데이트되었습니다.

* [Windows 터미널 설명서](/windows/terminal/)가 업데이트되었습니다.


지난 달에 최근 중요 업데이트가 확인된 항목 중 일부는 다음과 같습니다.
