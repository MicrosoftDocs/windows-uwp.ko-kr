## <a name="gameclipuritype"></a>GameClipUriType
 
| <b>열거자</b>| <b>설명</b>| 
| --- | --- | 
| 없음| 게임 클립 URI 유형 알 수 없는 되었거나 설정 되지 않았습니다.| 
| 원문 언어| 게임 클립 URI는 처리 되지 않은 형태로 원본 게임 클립의 위치입니다.|

| SmoothStreaming| 게임 클립 URI IIS 부드러운 스트리밍 URL입니다 (매니페스트 포인터).| 
| Ahls| 게임 클립 URL은 Apple Http 라이브 스트리밍 URL입니다.|
