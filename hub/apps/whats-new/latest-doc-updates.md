# <a name="latest-updates-to-the-windows-developer-docs"></a>Windows 개발자 문서의 최신 업데이트

Windows 개발자 문서는 정기적으로 개선된 최신 정보와 콘텐츠로 업데이트됩니다. 2021년 1월 15일의 변경 내용은 다음과 같습니다.

참고: Windows 10 빌드 19041(2004라고도 함)의 일부로 추가된 특정 API 목록은 [이 목록](/windows/uwp/whats-new/windows-10-build-19041-api-diff)을 참조하세요.

최신 Windows 개발자 설명서 뉴스를 보거나 의견 및 질문을 남길 수 있는 Twitter 핸들은 [@WindowsDocs](https://twitter.com/windowsdocs)입니다.

Microsoft 문서에 기여하는 방법에 대한 자세한 내용은 [기여자 가이드](https://docs.microsoft.com/contribute/)를 참조하세요.

이번 달의 주요 내용은 다음과 같습니다.

### <a name="new-content"></a>새로운 내용


* [WinUI 2.5](https://docs.microsoft.com/windows/apps/winui/winui2/)
* [PowerToys 설명서](https://docs.microsoft.com/windows/powertoys/)


* [IMsRdpClientNonScriptable6](https://docs.microsoft.com/windows/win32/termserv/imsrdpclientnonscriptable6)
* [IMsRdpClientNonScriptable7](https://docs.microsoft.com/windows/win32/termserv/imsrdpclientnonscriptable7)
* [IMsRdpCameraRedirConfigCollection](https://docs.microsoft.com/windows/win32/termserv/imsrdpcameraredirconfigcollection)
* [IMsRdpCameraRedirConfig](https://docs.microsoft.com/windows/win32/termserv/imsrdpcameraredirconfigcollection)
* [IMsRdpClipboard](https://docs.microsoft.com/windows/win32/termserv/imsrdpclipboard)

### <a name="updated-topics"></a>업데이트된 항목

* [이미지 토픽 캡처](https://docs.microsoft.com/windows/win32/gdi/capturing-an-image)
* [RASENTRY](https://docs.microsoft.com/previous-versions/windows/desktop/legacy/aa377274(v=vs.85))
* [Segoe MDL2 자산 아이콘](https://docs.microsoft.com/windows/uwp/design/style/segoe-ui-symbol-font)


### <a name="other-content-of-interest"></a>기타 내용



* 샘플 - C++/CX 샘플을 C++/WinRT로 지속적으로 업데이트하고 있습니다(예: [IDirect3DSurface Interface](https://docs.microsoft.com/uwp/api/windows.graphics.directx.direct3d11.idirect3dsurface)). 페이지 오른쪽 위에 있는 언어 선택기를 사용하여 C++/WinRT를 선택하세요.
